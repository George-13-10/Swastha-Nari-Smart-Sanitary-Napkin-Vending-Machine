# Swastha Nari – Smart Sanitary Napkin Vending Machine

Swastha Nari is an IoT-based Automatic Sanitary Napkin Vending Machine designed to provide hygienic and convenient access to sanitary napkins in schools, colleges, hospitals, railway stations, and public places.

The system supports both RFID-based authentication and UPI-based cashless payment. Once a valid RFID card is detected or a successful UPI transaction is confirmed, the Arduino-controlled dispensing mechanism activates a DC motor through an L298N motor driver to dispense a sanitary napkin automatically.

This project aims to improve menstrual hygiene accessibility while maintaining privacy, affordability, and 24/7 availability.
